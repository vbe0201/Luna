Client

class Client implements EventEmitterInterface (View source)

The generic Lavalink client. It does absolutely nothing for you on the Discord side.

The lavalink client implements automatic failover. That means, if a lavalink node unexpectedly disconnects, the client will automatically look for a new node and starts playing the track on it.
