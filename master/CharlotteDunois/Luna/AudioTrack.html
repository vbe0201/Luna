Constants

AUDIO_END_REASON_CONTINUE

Indicates whether a new track should be started on receiving the trackEnd event. If this is false, either this event is already triggered because another track started (REPLACED) or because the player is stopped (STOPPED, CLEANUP).

const AUDIO_END_REASON_CONTINUE = array(
    'FINISHED', 'LOAD_FAILED'
);
